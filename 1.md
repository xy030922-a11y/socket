


**C++/Qt 多线程下载器**

* 基于 **C++17 + Qt Widgets + SQLite** 开发桌面端多线程下载器，实现 HTTP/HTTPS 文件下载、暂停、继续、删除、进度显示和历史记录管理等功能。
* 采用分层架构设计，将 **UI 显示、下载任务管理、数据库操作、线程执行** 解耦，使用 `AppController` 统一协调各模块，提升代码可维护性。
* 使用 `QThread + Worker` 模式实现下载任务异步执行，避免 UI 阻塞；通过 Qt 信号槽机制实时更新下载进度、文件大小、下载速度和任务状态。
* 支持断点续传，下载过程使用 `.part` 临时文件保存数据，暂停后可根据已下载字节数通过 HTTP `Range` 继续下载。
* 使用 SQLite 持久化保存下载历史，支持启动时自动加载历史记录，并实现右键菜单、批量选择、批量暂停/开始/删除等交互功能。
